# Improved-PST-Pose-Estimation
An improved PST pose estimation algorithm designed for UAV visual navigation, distinguished by its incorporation of distortion compensation and multi-frame optical center consistency constraints.

This repository provides an implementation of a pose estimation algorithm based on Perspective Similar Triangles (PST), featuring the following improvements:

- Geometric distortion constraints and real-time correction
- Reformulate the algorithm structure
- Multi-frame Optical Center Consistency (MOCCC)


This method aims to enable robust and real-time visual navigation for UAVs.
